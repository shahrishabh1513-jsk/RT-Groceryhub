# 📌 About The Project

**RT GroceryHub** is a modern grocery shopping website designed to provide users with a smooth and easy online shopping experience.

The project focuses on:
- 🛒 Clean grocery product display
- 📱 Responsive user interface
- ⚡ Easy navigation
- 🎨 Attractive modern design
- 🧾 Organized product categories

This project showcases practical frontend web development skills and modern UI design concepts.

---

# 🚀 Features

- 🛍️ Grocery Product Listings

- 🎨 Modern UI Layout
- 🔍 Easy Product Browsing
- ⚡ Fast & Lightweight Website
- 🧾 Product Price & Quantity Display
- 🛒 User-Friendly Shopping Experience
